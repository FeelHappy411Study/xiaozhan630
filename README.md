肖战工作室及其粉丝的不可取行径档案
=========================
该文档建立于2020年7月1日，用于归档收录肖战工作室及其粉丝的不可取行为。

由于该团体行径过于拙劣，影响社会安定，法律却无法给予该团体应得的惩戒，故尽量收集相关证据，等待“迟来”的正义。

=========================
